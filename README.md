## Instalación

Desde la raíz del proyecto:

```bash
python -m venv .venv
source .venv/bin/activate
pip install opencv-python matplotlib numpy jupyter
```

## Uso

Activar el entorno virtual e iniciar Jupyter:

```bash
source .venv/bin/activate
jupyter notebook
```

Luego abrir el notebook correspondiente desde la interfaz:
- `parte1/Tarea2_Parte1.ipynb`
- `parte2/Tarea2_Parte2.ipynb`
