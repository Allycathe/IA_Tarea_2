# Tarea 2 — Procesamiento de Imágenes con OpenCV

## Requisitos

- Python 3.12
- Jupyter Notebook

## Estructura
````
.
├── parte1/
│   ├── imgs/                  # Imágenes de entrada
│   │   ├── img1.jpg
│   │   ├── img2.jpg
│   │   └── img3.jpg
│   ├── comparativa_img1.png   # Resultados generados
│   ├── comparativa_img2.png
│   ├── comparativa_img3.png
│   └── Tarea2_Parte1.ipynb    # Notebook parte 1
├── parte2/
│   └── Tarea2_Parte2.ipynb    # Notebook parte 2
├── .venv/                     # Entorno virtual
└── T2_IA2026-1.pdf            # Enunciado
````
## Instalación

Desde la raíz del proyecto:

```bash
python -m venv .venv
source .venv/bin/activate
pip install opencv-python matplotlib numpy jupyter
```

## Uso

Activar el entorno virtual e iniciar Jupyter:

```bash
source .venv/bin/activate
jupyter notebook
```

Luego abrir el notebook correspondiente desde la interfaz:
- `parte1/Tarea2_Parte1.ipynb`
- `parte2/Tarea2_Parte2.ipynb`
